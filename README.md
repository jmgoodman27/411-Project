# Podcasts App

Podcasts web application for my final class project at Penn State. 

### Built with
- Node.js/Express
- iTunes Search API
- Postgres
- Knex.js
- Sass
- Nunjucks
- Passport.js
- Gulp
- MVC architecture

### Functionality
- Search
- Browse 
- Log in / Create account
- Favorite, rate, and comment podcasts
- Search friends and view favorites
- Listen to podcasts

Deployed with Heroku.
